# Developmental links between play behavior and brain network integration

## About
This repository contains the code and data supporting the study *"Developmental links between play behavior and brain network integration"* submitted on *BioArxiv*.

## code

To reproduce Study1,2
- `Study1,2_infant_childhood.ipynb`: behavioral and demographic data
- `fc_Study1,2.Rmd`: brain functional connectivity
- `partclust_Study1,2.Rmd`: brain network measures (participation and clustering coefficient)

To reproduce Study3
- `Study3_school.ipynb`: behavioral and demographic data
- `fc_Study3.Rmd`: brain functional connectivity

## dataset
Includes all the data needed to run the code.
